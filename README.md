# Personal Portfolio

Un sitio web personal limpio y responsivo diseñado para mostrar mis proyectos, habilidades y trayectoria como desarrollador web.

**Live Demo:** [https://jdr93portfolio.vercel.app/](https://jdr93portfolio.vercel.app/)

## 🚀 Tecnologías Utilizadas

El proyecto está construido con tecnologías web estándar para garantizar ligereza y rendimiento:

* **HTML5:** Estructura semántica del contenido.
* **CSS3:** Diseño personalizado, Flexbox, CSS Grid y animaciones.
* **JavaScript (ES6+):** Lógica de interactividad y manipulación del DOM.

## 🛠️ Características

* **Diseño Responsive:** Optimizado para dispositivos móviles, tablets y escritorio.
* **Sección de Proyectos:** Galería detallada con enlaces a repositorios y demos.
* **Formulario de Contacto:** Funcional y validado.
* **Optimización SEO:** Estructura de etiquetas optimizada para buscadores.
* **Performance:** Carga rápida gracias al uso de código nativo sin frameworks pesados.
